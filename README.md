# VN Banking Portfolio Optimization

Dữ liệu và code tối ưu hóa danh mục đầu tư cổ phiếu ngân hàng Việt Nam  
theo phương pháp **Black-Litterman** kết hợp phân cụm chuỗi thời gian K-means.

## Dữ liệu
- `data/bank_monthly_close.csv` — Giá đóng cửa hàng tháng, 25 NHTM, 2015–2026
- `data/bank_monthly_mktcap_bn.csv` — Vốn hoá thị trường (tỷ VND), hàng tháng

## Ngân hàng
VCB, BID, CTG, MBB, TCB, ACB, VPB, HDB, VIB, TPB,  
STB, SHB, LPB, SSB, MSB, OCB, NAB, BAB, KLB, PGB,  
VAB, VBB, SGB, ABB, BVB
